Crash Demo (Information Disclosure)

  • ZeroDivisionError: Division by zero.
  • KeyError: Accessing a missing key in a dictionary.
  • TypeError: Invalid type operation.
  • Custom Exception: Raises a custom error.
  • HTTP 404/403/500: Triggers HTTP error responses.
  • Simulate Slow Response: Waits 5 seconds before responding.
  • MemoryError: Attempts to allocate a huge list.
  • OSError: Tries to open a non-existent file.
  • Infinite Loop: Simulates an infinite loop, but times out after 3 seconds.
Note: All Python exceptions are logged to error_demo.log in the project root.
